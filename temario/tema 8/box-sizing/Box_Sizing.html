Propiedad box-sizing

La propiedad box-sizing define cómo se calcula el ancho y el alto de una caja. Con content-box, el tamaño especificado se aplica solo al contenido, mientras que el padding y el border se suman.

box-sizing: content-box
Ancho total objetivo: 200px (marco rojo).
La caja interior se extenderá MÁS allá de este marco.
Esta caja tiene width: 200px y usa content-box.
El padding y el borde se suman al ancho total, por eso la caja es más ancha que el marco rojo.
box-sizing: border-box
Ancho total objetivo: 200px (marco rojo).
La caja interior encaja EXACTAMENTE dentro de este marco.
Esta caja tiene width: 200px y usa border-box.
El padding y el borde están incluidos en esos 200px, por eso encaja justo dentro del marco rojo.
