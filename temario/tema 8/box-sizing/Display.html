Propiedad display

La propiedad display define cómo se muestra un elemento en la página y cómo participa en el flujo del documento. Algunos valores comunes son block, inline, inline-block, none y flex.

display: block

Los elementos de tipo block ocupan todo el ancho disponible y generan un salto de línea antes y después.

display: inline

Los elementos de tipo inline se colocan dentro de una línea de texto y solo ocupan el espacio necesario para su contenido.

display: inline-block

Un elemento inline-block se comporta como un inline (puede estar en la misma línea que otros), pero permite definir ancho y alto como un bloque.
