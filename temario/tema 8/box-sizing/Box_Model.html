Modelo de caja (Box Model)

En CSS, cada elemento se representa como una caja. El modelo de caja está formado por: contenido, padding (relleno), border (borde) y margin (margen). El padding y el borde se suman al tamaño total de la caja, mientras que el margin es el espacio exterior que separa la caja de otros elementos. Ejemplo de Caja base con magin, border y padding.
